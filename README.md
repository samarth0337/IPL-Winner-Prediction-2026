# 🏏 IPL Winner Prediction (Machine Learning Project)

## 📌 Overview

This project predicts the probable winner of IPL matches using historical data and machine learning techniques. The model analyzes past match performance, team statistics, toss decisions, and venue conditions to estimate outcomes.

---

## 🎯 Objective

To build a machine learning model that predicts IPL match winners and identifies the most probable team to win the IPL season based on historical trends.

---

## 📂 Dataset

The project uses two datasets:

* `matches.csv` → Match-level data (teams, winner, toss, venue, etc.)
* `deliveries.csv` → Ball-by-ball data (used for deeper insights)

---

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

---

## 🧠 Machine Learning Model

* Algorithm Used: Random Forest Classifier
* Problem Type: Classification
* Target Variable: Match Winner

---

## 🔍 Features Considered

* Team 1 & Team 2
* Toss Winner
* Toss Decision
* Venue

---

## 📊 Key Insights

* Teams like Mumbai Indians and Chennai Super Kings show consistent dominance
* Toss decisions have a noticeable impact on match outcomes
* Historical performance strongly influences predictions

---

## 🏆 Final Prediction

Based on historical data analysis and model output:

👉 **Predicted IPL 2026 Winner: Mumbai Indians**
